Entendendo Comunicação Client x Server

(Voltar)
  1. História da Web

    Nesta aula inicia com uma introdução sobre a história da evolução dos primeiros computadores a partir do ano de 1940, os ditos computadores de 1a geração. Para ser mais específico em 1944 foi desenvolvido o Harvard Mark 1 do tipo eletromecânico com válvulas para fazer os cálculos.

    Envolvendo o contexto de Guerra Mundial , em 1946, foi criado o Colossus pela Alemanha para fazer criptoanalista e, também, o ENIAC pelos EUA.

    A título de curiosidade o termo bug surgiu nesta época, cunhado pela programadora Grace Murray Hopper, devido ao aquecimento das válvulas e atraia insetos que danificam os equipamentos ao aquecimento.

    Em seguida temos os computadores de 2a geração entre os anos de 1959 e 1965, mudando de válvulas para transistores diminuindo seu tamanho.

    Depois temos os de 3ageração entre os anos de 1965 e 1970, que utilizam circuitos integrados com transistores, capacitores e resistores e a partir de 1971 temos a 4a geração que perdura até hoje com uso de microprocessadores. Alguns dizem que já estamos na 5a geração devido a inteligência artificial, criptomoedas e metaverso, apesar das controvérsias.

    A internet é derivada da necessidade de comunicação eficaz e rápida, surgindo com o uso de telégrafo. Durante a década de 50 com a Guerra Fria, houve um fortíssimo investimento tecnológico para se obter vantagem neste confronto. O interessante a participação do Tim Berners-Lee para apresentar um viés que não fosse militar para utilizar toda a rede de internet durante sua estadia no CERN.

  2. O que são Clients?

    Client no linguajar web trata-se do dispositivo que consome a informação podendo ser o navegador da web ou aplicativo que o usuário está usando para acessar e interagir com a aplicação ou site. Por exemplo, se você estiver usando qualquer navegador para acessar um site ou aplicação web, o navegador é o client nesse caso.

    Navegadores da web são programas de software que permitem aos usuários acessar e visualizar páginas da web na internet. Também é citado que o W3C fornece sobre todos as tags HTML e em quais navegadores ele funciona.

    Já as aplicações web são programas de software que são acessados por meio de um navegador da web em vez de serem instalados diretamente no dispositivo do usuário. Essas aplicações são hospedadas em servidores remotos e podem ser acessadas por usuários de qualquer lugar com conexão à internet.

    Ao se tratar de dispositivos móveis, como smartphones e tablets, desempenham um papel crucial em nossas vidas atualmente devido a sua portabilidade e praticidade, sendo importante pensar em sua responsividade.

  3. O que são Servers?

    Servidores são computadores ou sistemas de computadores dedicados a fornecer serviços, recursos ou funcionalidades para outros dispositivos, programas ou usuários em uma rede. Eles desempenham um papel fundamental para o armazenamento de dados, processamento de dados, hospedagem de sites e aplicações web.

    Hardwares se destaca por ser o componete fisica e estrutural do servidor, enquanto que os softwares são ligados ao sistema operacional, web, monitoramente, rede e outros.

    Existem vários tipos de servidores, cada um projetado para atender a necessidades específicas de uma organização ou aplicação, por exemplo, servidores web, servidores de aplicação, servidores de banco de dados, servidores de arquivos e servidores de e-mail.

    A hospedagem de sites é um serviço que permite que indivíduos e empresas publiquem seus sites na internet. Os provedores de hospedagem de sites fornecem os recursos e a infraestrutura necessários para armazenar, servir e manter os arquivos de um site, garantindo que ele esteja acessível aos usuários na web, sendo eles públicos ou privados.

  4. Linguagens de Programação

    A linguagem de programação é uma linguagem formal projetada para expressar instruções que podem ser processadas por um computador. Ela fornece um conjunto de regras sintáticas e semânticas para escrever programas de computador, permitindo que os desenvolvedores comuniquem suas intenções para o computador de uma maneira compreensível.

    O Server-Side é a parte de uma aplicação ou sistema que é executada no servidor em vez de no dispositivo do usuário. No desenvolvimento web, o lado do servidor geralmente lida com a lógica de negócios, manipulação de dados e comunicação com o cliente.

    Ao se falar sobre Client-Side, no desenvolvimento web, client-side é a parte de uma aplicação ou sistema que é executada no navegador ou dispositivo do usuário.

    O HTML considerada uma linguagem de programação é uma linguagem de marcação usado para estruturar e organizar o conteúdo de uma página da web, como texto, imagens e outros elementos.
